Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Search Text Input
Search Button Text Input (Leave empty to use icon)
Search Results Page Content Selector Select the Search Result page
Query Parameter Name Input Default: 'query'
Fill search box with existing query Checkbox
Input Size Selection Select input size using bootstrap default class options
  • (Blank)
  • Large: 18px — .input-group-lg
  • Small: 12px — .input-group-sm
Alignment Selection Select alignment using bootstrap default class options — Nothing gets applied - this should be removed
  • (Blank)
  • Left — .pull-left
  • Right — .pull-right
Clear Float Checkbox
Contextual CSS Class Selection Select panel styles using bootstrap default class options
  • (Blank) — .btn-default
  • Danger — .btn-danger
  • Info — .btn-info
  • Primary — .btn-primary
  • Success — .btn-success
  • Warning — .btn-warning
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Default Input Size — 14px
