Usage

Provide contextual feedback messages for typical user actions using default bootstrap contextual classes (.danger-.primary-.info-.success-.warning).
Use the contextual class .primary to define custom alert styles in the Custom Stylesheet CSS.
Optionally, configure to add a dismiss button to close the alert and if the alert should show again in a certain days time or never show again.


More information at getbootstrap.com
http://getbootstrap.com/components/#alerts
