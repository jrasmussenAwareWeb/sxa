Tab Content

Allowed In

  • Tab Set (default)

Usage

Insert tab content items to create tabs in a Tab Set with additional Allowed Component options.

Example

Site Builder

Web

Tab Content Items

The Tab Content component is comprised of a tab title and tab body. Additional Allowed Components can be placed within the tab body.

Tab Title

Enter the tab title as text — entry here is rendered h3 in the panel-title section.


Tab Body Text

Enter tab body text — entry here is rendered in the panel-body section.
Note that this item uses the Rich Text (RTE) editor so additional HTML can be entered.


Tab Body Placeholder

Add additional Allowed Components — classes here will be relevant to the placed components.

Navigating Tab Content

To navigation between tab content items, click on the area surround the title but not in the title area.
