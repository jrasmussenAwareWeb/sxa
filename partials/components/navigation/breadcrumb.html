Breadcrumb

Allowed In

  • Accordion
  • Container
  • Header
  • Row
  • Row Advanced
  • Tabset

Usage

Use Breadcrumb to indicate the current location within the navigational hierarchy.


More information at getbootstrap.com
http://getbootstrap.com/components/#breadcrumbs

Examples - Contextual Classes

(default)
.text-danger
.text-info

.text-primary
.text-success
.text-warning

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Breadcrumb Title Input

titles are managed within the page content Breadcrumb attributes in the Content Editor

Hide in Breadcrumb Checkbox Check to hide the Breadcrumb Title
Contextual CSS Class Selection Select breadcrumb title style using bootstrap default class options
  • Danger — .text-danger
  • Info — .text-info
  • Primary — .text-primary
  • Success — .text-success
  • Warning — .text-warning
Breadcrumb CSS Class Input Add CSS class
Breadcrumb Style Input Add CSS styles (inline)

Breadcrumb Attributes

Enter the breadcrumb title for the page — optionally use FontAwesome HTML as an alternative to text.

For example, navigate to the Home page within the Content Editor and enter the code below.

<i class="fa fa-home"></i>

Allowed Components

None

Code Sample

<div id="breadcrumb" class="" style="">
    <ol class="breadcrumb">
        <li>
            <a href="/Components" class="" style="">Components</a> <!--contextual classes applied here-->
        </li>
        <li>
            <a href="/Components/Navigation" class="" style="">Navigation</a>
        </li>
        <li class="text-info" style="">Breadcrumb</li>
    </ol>
</div>

Bootstrap Classes

.breadcrumb {
    list-style: outside none none;
}
.breadcrumb {
    background-color: #f5f5f5;
    border-radius: 4px;
    list-style: outside none none;
    margin: 15px 0;
    padding: 8px 15px;
}
/*contextual classes*/
.text-danger, .jumbotron h1.text-danger {
    color: #d9534f;
}
.text-info, .jumbotron h1.text-info {
    color: #2aabb9;
}
.text-primary, .jumbotron h1.text-primary {
    color: #0785a4;
}
.text-success, .jumbotron h1.text-success {
    color: #809f3d;
}
.text-warning, .jumbotron h1.text-warning {
    color: #e36526;
}

Tips & Troubleshooting

Restrictions & Dependencies

When using Breadcrumb, a separate component is not created. Therefore, only one breadcrumb can be used throughout the site.
