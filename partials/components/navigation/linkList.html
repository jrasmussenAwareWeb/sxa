Footer Navigation

Allowed In

  • Row
  • Row Advanced

Includes

  • Navigation Title

Usage

Use Footer Navigation to add additional content tree navigation sections in the footer.

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Column Header Item Content Selector Select the top level item for the navigation root
Column Header Title Input Enter column header title — required for mobile
Sub Navigation Levels Selection Select None or a number for how many subnavigation levels to show
  • None
  • 1
  • 2
  • 3
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Footer Navigation Items

Modify Footer Navigation Items by clicking into the Navigation Title area.

Content Group Options - Navigation Title


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Navigation Title Input Enter the link label text
Hide in Navigation Checkbox Check to hide in the Footer Navigation
Navigation Root Checkbox Check to make item navigation root i.e. parent

Tips & Troubleshooting

If planning to use Footer Navigation, note the mobile design renders as a toggle menu. Keep this in mind when planning other footer column items.

Restrictions & Dependencies

If placing within Row or Row Advanced, add the following correction to the Custom Stylesheet. 
@media(min-width:767px) {
.footer ul.collapse {
display:block;
visibility:visible;
}
}
