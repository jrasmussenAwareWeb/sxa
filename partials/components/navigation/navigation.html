Navbar Alternate

Allowed In

  • Container
  • Header

Includes

  • Navbar Component
  • Navbar Logo
  • Navbar Primary Nav

Usage

Use Navbar Alternate for included Navbar Logo, Navbar Primary Nav, and Navbar Component (for additional component options).
Navbar Alternate can only be placed in Header or Container.

For other placement (within Row or Row Advanced) or when an included Navbar Primary Nav is not required, use Navbar. Navbar Primary Nav can be placed within Navbar Component, but is not included by default.
