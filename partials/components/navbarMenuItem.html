Content Group Options

OR

To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Label Input
Link Content Selector
Modal Window Target Content Selector
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Note that the Content Group Options are the same whether you click on Link OR  Label

Linking to a Modal Window Using Modal Window Target

In the Content Group options select Modal Window Target and navigate, then select the Modal Window component.
