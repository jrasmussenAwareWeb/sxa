Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Button Text Input
Button Link Content Tree Selector
Button Size Selection Select alert styles using bootstrap default class options
  • Large Button — .btn-lg
  • Small Button — .btn-sm
  • Extra Small Button — .btn-xs
Full Width Button Checkbox Check this if you want the button to fill the width of the component
Display as Hyperlink Checkbox
Modal Window Target Content Tree Selector
Contextual CSS Class Selection Select alert styles using bootstrap default class options
  • Danger — .btn-danger
  • Info — .btn-info
  • Primary — .btn-primary
  • Success — .btn-success
  • Warning — .btn-warning
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)
