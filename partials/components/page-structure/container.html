Usage

Bootstrap requires a containing element to wrap site contents and house the grid system. Note that containers are not nestable.
Use .container for a responsive fixed width container. Use .container-fluid for a full width container.


More information at getbootstrap.com
http://getbootstrap.com/css/#overview-container

Tips & Troubleshooting

Add a correction to the custom stylesheet if using any fluid container(s).

.container-fluid 
.container-fluid {
    padding-left: 0;
    padding-right: 0;
}
