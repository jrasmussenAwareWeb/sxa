Panel

Allowed In

  • Accordion
  • Billboard Slide (Caption)
  • Container
  • Modal Window
  • Row
  • Row Advanced
  • Tabset

Usage

Use Panel for a bordered box section with padded body content and an optional panel title and footer. Additional Allowed Components can be added to the panel body.


More information at getbootstrap.com
http://getbootstrap.com/components/#panels

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Title Input Enter the panel title — for no title, leave blank
Title Link Content Selector Select a link for the title — leave blank for no linking
Footer Rich Text (RTE)
Contextual CSS Class Selection Select panel styles using bootstrap default class options
  • (Blank) — .panel-default
  • Danger — .panel-danger
  • Info — .panel-info
  • Primary — .panel-primary
  • Success — .panel-success
  • Warning — .panel-warning
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Code Samples

    <div class="panel-heading">
        <h3 class="panel-title">Title</h3>
    </div>
    <div class="panel-body">

    </div>
    <div class="panel-footer">

    </div>
