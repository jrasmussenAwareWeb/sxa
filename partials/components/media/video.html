Video Player

Allowed In

  • Accordion
  • Container
  • Modal Window
  • Panel
  • Row
  • Row Advanced

Usage

Use to embed an HTML5 video (MP4, OGG, WebM) uploaded to the Media Library.


More information
http://diveintohtml5.info/video.html
https://developer.mozilla.org/en-US/docs/Web/HTML/Supported_media_formats

Examples


Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
MP4 Video Media Library Selector Select target MP4 Video file from the Media Library
OGG Video Media Library Selector Select target OGG Video file from the Media Library
WebM Video Media Library Selector Select target WebM Video file from the Media Library
Video Aspect Ratio Selection Select aspect ratio the video should display at — (if aspect ratio is set, Height and Width Settings will be over-ridden with the aspect ratio).
  • Aspect Ratio 16x9 (16:9)
  • Aspect Ratio 4x3 4:3
Video Player Autoplay Checkbox Check to have the video autoplay on page thumbnail
Video Player Loop Checkbox Check to have the video loop
Video Player Height Input Enter video player desired height — (if aspect ratio has a value, width settings will not function)
Video Player Width Input Enter video player desired width — (if aspect ratio has a value, width settings will not function)
Video Player Poster Image Media Library Selector Select thumbnail image — (displays prior to hitting the play button)
Video Player Not Support Text Input Enter text for Video Not Supported Message — (default - "Your browser does not support the HTML5 video tag.")
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)
