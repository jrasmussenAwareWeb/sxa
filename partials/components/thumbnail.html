Usage

Extend Bootstrap's grid system with the thumbnail component to easily display grids of images, videos, text, and more.


More information at getbootstrap.com
http://getbootstrap.com/components/#thumbnails

Thumbnail Items

The Thumbnail component is comprised of a Title with optional linking, Body, Image (optional link), and Body Text Link (optional).

Thumbnail Title

Enter the Thumbnail Title as text — entry here is rendered in the caption h3 section.


Thumbnail Body

Enter Thumbnail Body text — entry here is rendered in the caption section.
Note that this item uses the Rich Text Editor (RTE) so additional HTML can be entered.


Thumbnail Body Text Link

Select Thumbnail Body Text Link


Thumbnail Image

Select Thumbnail Image

Thumbnail Image Select

To select an image, click the Choose an Image icon.

Thumbnail Image Edit

To edit an image, click the Modify Image Appearance icon.


Image Properties
Option Data Type Function
Text Input Alternate Text
Width Input Set the width of the image if they are different than the dimensions of the original
Height Input Set the height of the image if they are different than the dimensions of the original
Keep Aspect Ratio Checkbox
Horizontal space Input Set the horizontal space around the image
Vertical Space Input Set the vertical space around the image

Thumbnail Image Delete

To delete an image, click the Remove the Image icon.
