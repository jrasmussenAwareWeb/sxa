Usage

Use Row Advanced to create horizontal groups of columns when more flexibility on column size options for breakpoints is required.
Selection options are available for Multiple Breakpoints, Single Breakpoint - sm, and Single Breakpoint - xs as listed in Content Group Options.
Additional column classes can be added — see Adding Column Sizes for more information.

When you need only one breakpoint for mobile ( < 768px ), use Row.
