Modal Window

Allowed In

  • Accordion
  • Row
  • Row Advanced

Usage

Use Modal Window to display additional information or functionality without taking the user away from the page.
For example, a disclaimer or privacy policy link located in the footer or a navigation or other link to a register or login form.


More information at getbootstrap.com
http://getbootstrap.com/javascript/#modals

Linking to a Modal Window — Using ModalWindowTarget

A Modal Window can be linked to from Navbar Primary Nav or Navbar Utility Nav within Navbar Menu Item.

In the Content Group options select Modal Window Target and navigate, then select the Modal Window component.


Linking to a Modal Window — Manually

Use a Rich Text Editor or HTML component with the below sample code. 

<a href="#" data-toggle="modal" data-target="#ModalWindowLogin">Login</a>


Within the Modal Window, enter the Modal Title, the Modal Button Close Text (if any), and the Modal Data Target KeyName.
The Modal Window Target Keyname is the data-target that needs included in the link to the modal.

Example: <a href="#" data-toggle="modal" data-target="#ModalWindowLogin">Login</a>
