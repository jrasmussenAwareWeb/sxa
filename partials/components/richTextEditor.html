Usage

Use Rich Text Editor for complex sections where you require WYSIWYG ("What You See Is What You Get") editor capabilities such as font style and formatting options, images, tables, lists, and more.

Rich Text Editor Overview

Toolbar

Design

Select the Design tab for WYSIWYG styling using the Rich Text Editor toolbar.


HTML

Select the HTML tab for standard HTML editing — search is the only toolbar option available.
