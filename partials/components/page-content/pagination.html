Pagination - Pager

Allowed In

  • Container
  • Row
  • Row Advanced

Usage

Use Pagination - Pager for previous and next buttons on a section of or all pages.

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Start Item Content Selector Select Start (parent) Item — renders to all children under parent.
Search All Descendants Checkbox
Tags Tag Selector
Max Items to Display Input Set to 0 to return all results
Sort Order Selection Select the sort order using default options
  • Alphabetical
  • Content Tree Order
  • Newest
  • Oldest
  • Recently Updated
Align Buttons to Sides of Page Checkbox Check to float Previous button left and Next button right
Previous Button Text override Input Enter to text to use instead of 'Previous'
Next Button Text override Input Enter to text to use instead of 'Next'
CSS ID Input Enter ID
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Allowed Components

None

Code Sample

<nav id="" class="" style="">
    <ul class="pager">
        <li class="previous">
            <a href="/GenericSampleCompany/About/OurTeam"><span aria-hidden="true">?</span> Previous</a>
        </li>
        <li class="next">
            <a href="/GenericSampleCompany/About/RequestAppointment">Next <span aria-hidden="true">?</span></a>
        </li>

    </ul>
</nav>

Classes

.pager {
    list-style: outside none none;
    margin: 20px 0;
    padding-left: 0;
    text-align: center;
}
.pager li {
    display: inline;
}
.pager li > a, .pager li > span {
    background-color: #fff;
    border: 1px solid #ddd;
    border-radius: 15px;
    display: inline-block;
    padding: 5px 14px;
}
.pager .previous > a,
.pager .previous > span {
    float: left;
}
.pager .next > a,
.pager .next > span {
    float: right;
}
