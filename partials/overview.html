Overview

What are the core features?

  • Template Engine - Create as many templates as you need without going back to a developer
  • Drag-n-drop Page Assembly with Reusable OOB Components - Over 30+ pre-built components that you can use to build any page
  • Dynamic Wireframes - Create wireframes directly from within Sitecore, allowing design and content entry to start once UX is complete
  • Pluggable Themes - Completed design (one or more) can be imported as a theme (or themes) and easily switched at any time
  • Creative Exchange - Seperate UA from design; wires can be exported as a site in static HTML, JS, and CSS
  • Mobile First - Design for mobile without non-deliverability issues
  • Parallel Production - Reduce time to market

Parallel Production

The most important overall feature of XsA is the potential reduction of time to market via Parallel Production methodologies provided by the extensible and customizable toolset.
